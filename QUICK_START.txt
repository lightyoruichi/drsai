✅ Page 1: Upload Page (/)
   - Upload dental scans (images/PDFs)
   - Initial diagnosis text input
   - Drag & drop support
   - Beautiful modern UI

✅ Page 2: Processing Page (/processing)
   - Animated loading states
   - Progress indicators
   - Calls OpenRouter API
   - Auto-redirects when done

✅ Page 3: Brief Results (/results/[id])
   - Shows uploaded scan
   - AI diagnosis with confidence score
   - Compares with initial diagnosis (right/wrong/additional)
   - Forever shareable link
   - Quick recommendations

✅ Page 4: Full Results (/results/[id]/full)
   - Interactive 32-tooth dental chart
   - Tooth-by-tooth analysis
   - Complete detailed findings
   - Color-coded severity levels
   - Print/download options

🚀 TO GET STARTED:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   $ npm install

2. Get OpenRouter API key:
   → Visit https://openrouter.ai/keys
   → Create new key
   → Copy it (starts with sk-or-v1-...)

3. Create .env file:
   $ cp .env.example .env
   
4. Edit .env and add your key:
   OPENROUTER_API_KEY=sk-or-v1-your-actual-key-here

5. Start dev server:
   $ npm run dev

6. Open browser:
   → http://localhost:4321

7. Test it:
   → Upload any image (dental scan preferred)
   → Enter initial diagnosis
   → Wait 15-30 seconds
   → View results!
